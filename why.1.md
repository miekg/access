# NAME

why - explain why a user can or cannot access a file

# SYNOPSIS

**why** [**OPTIONS**] **USERNAME** [*FILE...*]

# DESCRIPTION

# OPTIONS

`-ast`

:  print abstract syntax tree and exit

`-fragment`

:  don't create a full document
